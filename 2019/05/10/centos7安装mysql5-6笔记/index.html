centos7安装mysql5.6笔记

2019-05-10 mysql linux, mysql 0 评论 字数统计: 245(字) 阅读时长: 1(分)

在centos7中使用yum命令安装mysql的时候，默认安装的是mariaDB，那么该如何安装指定版本的mysql呢？接下来，将逐步介绍centos7从零搭建mysql的过程。（为了避免出现权限方面的问题，这里统一采用root账号操作，并且需要关闭防火墙或者指定需要开放的规则）

下载mysql的repo源，这里下载的是5.6版本

`curl -O http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm`

安装源

`rpm -ivh mysql-community-release-el7-5.noarch.rpm`

安装mysql服务端

`yum -y install mysql-server`

启动mysql服务

`systemctl start mysql`

修改root密码

`mysqladmin password 自己要设置的密码`

登录

`mysql -u root -p`

创建feature用户，并分配远程访问的权限

## 创建用户feature，指定密码
create user feature identified by 'feature用户的密码';
## 授权，并设置可以远程访问
grant all privileges on *.* to feature@'%' identified by 'feature用户的密码'; 
## 刷新权限
flush privileges;
